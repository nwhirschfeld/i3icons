# i3icons
native deamon to rename workspaces with fontawesome-icons based on open applications


## Getting started

### Building from source

  ~~~ sh
  $ git clone --recursive https://github.com/nwhirschfeld/i3icons.git 
  $ mkdir i3icons/build
  $ cd i3icons/build
  $ cmake ../src
  $ sudo make install
  ~~~
